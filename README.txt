Per eseguire il codice "code.py" è necessario scaricare il dataset Avila, disponibile al seguente URL:

http://archive.ics.uci.edu/ml/machine-learning-databases/00459/

Decompresso l'archivio, i due file "avila-tr.txt" e "avila-ts.txt" vanno spostati nella STESSA cartella in cui si trova il codice "code.py".

Da lì si può eseguire con Python3 lo script.

Vengono poi generati nella stessa cartella i due .csv che contengono i risultati.


Il file "Relazione.pdf" spiega come il codice agisce e riporta i risultati ottenuti, in relazione all'articolo di riferimento.

L'articolo di riferimento (De Stefano et al. 2018) è disponibile all' URL:

https://www.sciencedirect.com/science/article/abs/pii/S0952197618300721
